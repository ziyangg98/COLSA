# Compute Log-Likelihood for 'colsa' Objects

This function retrieves the log-likelihood value from an object of class
'colsa'.

## Usage

``` r
# S3 method for class 'colsa'
logLik(object, ...)
```

## Arguments

- object:

  An object of class 'colsa'.

- ...:

  Additional arguments (currently unused).

## Value

The log-likelihood value stored in the 'colsa' object.
