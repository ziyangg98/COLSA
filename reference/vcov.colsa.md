# Compute Variance-Covariance Matrix for 'colsa' Objects

This function computes the variance-covariance matrix for objects of
class `"colsa"`.

## Usage

``` r
# S3 method for class 'colsa'
vcov(object, ...)
```

## Arguments

- object:

  An object of class `"colsa"`.

- ...:

  Additional arguments (currently not used).

## Value

A matrix representing the variance-covariance matrix of the model
parameters.
